# Student Management System

A console-based Student Management System built using Java, JDBC, and MySQL.

## Features
- Add Student
- View Students
- Update Student
- Delete Student
- Input Validation
- MySQL Database Connectivity

## Technologies Used
- Java
- OOP
- JDBC
- MySQL
- Git & GitHub

## Project Structure

StudentManagementSystem
│
├── Student.java
├── DBConnection.java
├── StudentDAO.java
├── Main.java
└── lib
    └── mysql-connector-j.jar

## How to Run

1. Clone the repository
2. Open project in VS Code
3. Add MySQL JDBC Driver
4. Compile:

```bash
javac -cp ".;lib/mysql-connector-j.jar" *.java
```

5. Run:

```bash
java -cp ".;lib/mysql-connector-j.jar" Main
```
